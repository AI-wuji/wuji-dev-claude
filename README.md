### 核心理念

> **初期对齐，全程自主。静默联动，无极无感。**

| 理念 | 说明 |
|------|------|
| 🎯 初期对齐 | 需求不清晰时一次性问完，对齐后独立跑到底 |
| 🤫 全程自主 | 搜索→编码→测试→修复→打磨→交付，中间不打扰无极 |
| 🔗 静默联动 | UI任务自动走Impeccable，思维蒸馏自动走女娲，无需手动切换 |
| 🛡️ 安全第一 | 打靶场沙盒测试 + 四级权限 + 八条红线 + 改前备份 |
| 🧬 错误DNA | 每次Bug修复自动记录，相似度检测，三振强制根因分析 |
| 🔥 迭代熔断 | 输出重复>85%自动停，≤3轮上限 |

## 🛡️ 安全体系

### 四级权限

| 级别 | 范围 | 处理 |
|------|------|------|
| 🟢 无害 | 读文件、搜索、分析 | 直接执行 |
| 🟡 本地 | 写文件、运行测试 | 直接执行 |
| 🟠 中等 | 安装依赖、改配置、git操作 | 确认后执行 |
| 🔴 高危 | 删文件、force push、改环境变量 | **必须无极确认** |

### 打靶场沙盒

```
python scripts/target_range.py code <file>          # 扫描代码安全
python scripts/target_range.py plugin <dir>         # 扫描ComfyUI插件
python scripts/target_range.py dependency <name>    # 检查依赖
python scripts/target_range.py config <file>        # 检查配置
python scripts/target_range.py permission <描述>    # 评估权限等级
```
